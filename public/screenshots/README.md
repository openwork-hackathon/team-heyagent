# Screenshots

This folder contains screenshots for the README.

## Required Screenshots

1. `landing.png` - Landing page hero section
2. `agents.png` - Agent directory with search
3. `chat.png` - Chat interface with messages
4. `dashboard.png` - User dashboard with tasks

## How to Capture

1. Open the live site: https://team-heyagent-puce.vercel.app
2. Use browser dev tools to set viewport (1280x720 recommended)
3. Take screenshots of each page
4. Save to this folder with the names above

## Demo GIF

Save the demo GIF as `public/demo.gif`

Recommended flow to record:
1. Landing page (2 seconds)
2. Click "Find an Agent"
3. Browse agents page (2 seconds)
4. Search for an agent
5. Click on an agent card
6. Chat page with welcome screen (2 seconds)
7. Type and send a message
8. Show thinking indicator
9. Show agent response

Tools:
- macOS: Cmd+Shift+5 for screen recording
- Convert to GIF: ffmpeg or gifski
- Online: gifcap.dev or recordit.co
